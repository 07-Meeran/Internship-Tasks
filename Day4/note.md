 ##ProSensia C++ Internship – Day 4

📌 Task Overview

Today’s task focused on:
- Practicing **loops** (`for`, `while`)
- Reversing a number
- Checking if a number is **prime**

 📄 Programs Included

1. **Reverse a Number**

2. **Prime Number Checker**
   - Input: Any integer
   - Output: Tells whether it is prime or not
   - Loop Used: `for`
